Intro:

In today’s era, the internet and smartphones have become an integral part of our lives. We use our phones for almost all of our day to day tasks. Earlier we used to shop at malls, queue up in front of banks, make our travel bookings at a travel agency and wait for a cab to show up. But now, we can do all this in the comfort of our homes using our smart phones. All of this is possible because a high-level programming language called Java.

First of all What is Java? Why is it so important in the computer world?

Java is an object-oriented programming language that is used in a distributed environment on the internet. It is a high-level language that is easy to read and understand. Java is popular used in console, web, mobile applications and game development. It is not only used in computers and mobiles but even in electronic devices like televisions, washing machines, elevators. Java been invented by James Gosling and Sun microsystems in 1991. It’s important in the computer world because you can use the same code anywhere anytime.

To start coding and programming in Java, a programmer must have two main things installed on his device that will allow him to access everything Java language has to offer. First, what programmers call JDK; a JDK is the Java Development Kit that provides you with the right programming environment and adjusts some of your device's settings so that it can handle Java control. Secondly to download a text editor program there are multi programs you can use such as: Notepad, Sublime, Visual Studio and Atom. They allow the user to type the code and save it. In this manual you will use Visual Studio.

 


What is JDK:

Java Development Kit is the key fundamental software component that our computers need to have in order to be able to program in java using any operating system such as Windows, MacOS or Linux, or as programmers like to call it JDK. Also, is the environment that is responsible of developing (typing code in the text editor), compiling (check if there any errors in your code) and executing (run the program).  I’m going to show how to download it on MacOS in this manual.

Start Programming


You now have our two major components, which it’s ready to start and build our first small Java project. The project is aimed to build a program that can multiply two numbers together.

* Note: Make sure you write everything as it shows because Java is a case sensitive.
* Note: Most lines will have to end with semi colon ;


1. To start coding you have to press on new file. Then will take you to a blank black page.


2.You will need a class and a main method inside our editor in every Java program which will act as a code scope.



3. You will need a Scanner to ask the user to give his input on what the numbers he wants to multiply, so you have to type on the first line: import java.util.Scanner;
In the main method you build the scanner to be able to read user’s input by using the keyboard.
Scanner scan = new Scanner (System.in).



4. Now you have to type System.out.println(“Enter first number”); This will show whatever is in the quotation when yourun the program. After that you type int num1= scan.nextInt(); This method reads the number the user provide.

5. you redo the same steps from step 4 to do the second number, but you have to change the names to num2. System.out.println(“Enter second number”);
int num1= scan.nextInt();


6. In this step the two inputs get multiplied that the user have type in.
int answer = num1*num2;
System.out.println(“Answer:  ” + answer);

7.This is the final step that you  have to save the code on your desktop and name the code as the class name and followed with .java (Multiply.java)

Run the program

Running a java program requires simple steps for the user to follow. Under the hood, though, a sophisticated process is underway. The method converts the java code to binary code consisting only from 0's and 1's, which is kind of human readable.

Steps How to Run the Program

1. First step that you have to open the terminal. You have to type cd Desktop then press enter.
2. Then you type java Multiply.java this step will check if there any errors in your code.
3. Last step to run the code you type java Multiply. It will ask the user to enter two Numbers and after entering them it will give you the answer.
